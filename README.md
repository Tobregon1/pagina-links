# 🚀 Tobias | Links - Landing Page Premium

Una landing page de enlaces (estilo Linktree) moderna, minimalista y de alto rendimiento, diseñada para centralizar redes sociales, recursos educativos y proyectos personales con una estética **premium**.

## ✨ Características Principales

- **💎 Diseño Glassmorphism**: Interfaz moderna con efectos de desenfoque y transparencias sutiles.
- **🌓 Modo Inteligente**: Soporte completo para temas Claro y Oscuro con persistencia automática (recuerda la elección del usuario).
- **📱 Mobile-First**: Optimizada para una visualización perfecta en dispositivos móviles, simulando la experiencia de una App nativa.
- **🔗 Compartir Inteligente**: Botón de compartir integrado que utiliza la API nativa de dispositivos móviles o copia el enlace al portapapeles con un aviso (toast) sutil.
- **🎨 Branding Dinámico**: Iconos de redes sociales que adoptan los colores de su marca y muestran etiquetas informativas al pasar el mouse.
- **📈 Optimización SEO & OG**: Configurada para que el enlace se vea profesional al ser compartido en WhatsApp, Instagram o Facebook (título, descripción e imagen).

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura semántica y accesible.
- **CSS3**: Diseño responsivo, variables CSS, animaciones fluidas y efectos de vidrio.
- **JavaScript (Vanilla)**: Lógica de cambio de tema, persistencia en `localStorage` y funcionalidad de compartir.
- **FontAwesome**: Iconografía profesional.
